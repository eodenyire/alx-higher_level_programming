# SQL - Introduction

This directory contains scripts and solutions for the SQL introduction tasks in the ALX Software Engineering Program.

## Project Overview

The aim of this project is to introduce the basics of SQL and MySQL. The tasks cover a variety of fundamental concepts and operations in SQL, such as creating databases, tables, inserting data, updating records, and performing queries.

## Learning Objectives

By the end of this project, you should be able to explain the following concepts without assistance:

- What is a database
- What is a relational database
- What does SQL stand for
- What is MySQL
- How to create a database in MySQL
- What does DDL and DML stand for
- How to CREATE or ALTER a table
- How to SELECT data from a table
- How to INSERT, UPDATE, or DELETE data
- What are subqueries
- How to use MySQL functions

## Project Requirements

- All scripts are executed on Ubuntu 20.04 LTS using MySQL 8.0.
- Each SQL query is preceded by a comment explaining the task.
- All SQL keywords are written in uppercase.
- A `README.md` file is included in the root of the project directory.
- Scripts must end with a new line and start with a comment describing the task.

## Tasks

1. **0-list_databases.sql** - Lists all databases on the MySQL server.
2. **1-create_database_if_missing.sql** - Creates the database `hbtn_0c_0` if it doesn't exist.
3. **2-remove_database.sql** - Deletes the database `hbtn_0c_0` if it exists.
4. **3-list_tables.sql** - Lists all tables of a database passed as an argument.
5. **4-first_table.sql** - Creates a table called `first_table` with `id` and `name` columns.
6. **5-full_table.sql** - Prints the full description of the table `first_table`.
7. **6-list_values.sql** - Lists all rows of the table `first_table`.
8. **7-insert_value.sql** - Inserts a new row into the table `first_table`.
9. **8-count_89.sql** - Counts the records with `id = 89` in the table `first_table`.
10. **9-full_creation.sql** - Creates a table `second_table` and adds multiple rows.
11. **10-top_score.sql** - Lists all records from `second_table` ordered by score.
12. **11-best_score.sql** - Lists records with score >= 10 from `second_table`.
13. **12-no_cheating.sql** - Updates the score of `Bob` to 10 in `second_table`.
14. **13-change_class.sql** - Removes all records with a score <= 5 in `second_table`.
15. **14-average.sql** - Computes the average score of all records in `second_table`.
16. **15-groups.sql** - Lists the number of records with the same score in `second_table`.
17. **16-no_link.sql** - Lists all records from `second_table` without rows with empty `name`.

## Advanced Tasks

18. **100-move_to_utf8.sql** - Converts the `hbtn_0c_0` database to UTF8.
19. **101-avg_temperatures.sql** - Displays the average temperature by city ordered by temperature.
20. **102-top_city.sql** - Displays the top 3 cities' temperatures during July and August.
21. **103-max_state.sql** - Displays the max temperature of each state ordered by state name.
